# Bind Attribute

![Events](/static/images/ditto.webp)

```html
<img
    data-bind-width="$w"
    data-bind-height="$h"
    src="https://via.placeholder.com/350x150"
/>
```

Allows you to bind to any attribute of an element works like `data-text` but for any attributes.
